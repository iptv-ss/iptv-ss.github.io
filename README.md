# StarShare IPTV Website

Static marketing website for StarShare IPTV, built with plain HTML, CSS, and JavaScript, and hosted on GitHub Pages.

## Pages

- `index.html` - home page with features, pricing, downloads, and FAQ
- `tutorial.html` - setup guide and troubleshooting tips

## Tech Stack

- HTML5
- CSS3 (`styles.css`)
- Vanilla JavaScript (`script.js`)

## SEO Setup

The site includes:

- Page-specific title and meta description tags
- Canonical URLs
- Open Graph and Twitter meta tags
- Structured data (JSON-LD): `Organization`, `WebSite`, `FAQPage`, `HowTo`, `BreadcrumbList`
- `robots.txt`
- `sitemap.xml`

## Deployment

Hosted on GitHub Pages at:

- [https://iptv-ss.github.io/](https://iptv-ss.github.io/)

## Maintenance Notes

- Keep internal links consistent (use `tutorial.html` path)
- Update `sitemap.xml` when adding new pages
- Re-submit sitemap in Google Search Console after major updates
